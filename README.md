## Installation

install the basic features:

`go get -v github.com/elvinchan/rpcx/...`


If you want to use `quic` registry, use those tags to `go get` „ÄÅ `go build` or `go run`. For example, if you want to use all features, you can:

```sh
go get -v -tags "quic" github.com/elvinchan/rpcx/...
```
